# Credit-Card-Financial-Dashboard
Interactive Power BI dashboard analyzing credit card transactions, revenue trends, and customer spending behavior with actionable business insights.
## 📌 Project Overview
This project provides a comprehensive analysis of credit card financial operations and customer demographics. By integrating transaction and customer data, the dashboard identifies revenue trends, monitors key financial metrics, and analyzes spending behavior. The workflow follows the complete data cycle: **Data Extraction**, **Data Cleaning**, **Data Modeling**, and **Interactive Visualization**.

---

## 🎯 Problem Statement
Financial institutions often face challenges in tracking real-time performance and understanding customer risk factors. Key challenges addressed in this project:
* **Revenue Monitoring:** Difficulty in tracking weekly and quarterly revenue growth.
* **Customer Segmentation:** Lack of clarity on which demographic segments contribute most to the revenue.
* **Risk Management:** Difficulty in identifying delinquency rates and high-risk accounts to minimize financial losses.

This project solves these issues by automating the calculation of key KPIs, providing stakeholders with actionable insights.

---

## 📊 Dataset Information
The project utilizes two primary datasets:
* **Domain:** Banking & Finance
* **Files:** `credit_card.csv` (Transaction details) and `customer.csv` (Demographic details).
* **Data Points:** Over 10,000+ records.
* **Key Columns:**
    * **Transactions:** `Annual_Fees`, `Total_Trans_Amt`, `Interest_Earned`, `Card_Category`.
    * **Customer Profile:** `Customer_Age`, `Gender`, `Income`, `Education_Level`, `Customer_Job`.

---

## 🛠️ Tools & Technologies Used
* **Power BI Desktop:** For data modeling and creating interactive visualizations.
* **DAX (Data Analysis Expressions):** To create custom measures and calculated columns.
* **Python/Excel:** Used for data cleaning, handling missing values, and formatting.
* **GitHub:** For project documentation and version control.
